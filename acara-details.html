Pelatihan Website - HTML Dasar

Di sesi ini, peserta dikenalkan dengan struktur dasar HTML sebagai fondasi utama dalam membangun website. Mulai dari membuat halaman sederhana, menata elemen teks, hingga memahami tag-tag penting seperti head, body , h1 sampai a, semua dijelaskan dengan praktik langsung. Cocok untuk pemula yang ingin memahami dunia web dari awal.

Peserta juga diajak langsung membuat mini project berupa halaman profil sederhana untuk melatih pemahaman materi. Dengan pendekatan belajar yang interaktif dan hands-on, sesi ini jadi langkah pertama yang solid untuk mulai masuk ke dunia web development.

Highlight Acara

  • Pengenalan dasar HTML dan struktur dokumen web
  • Praktik membuat halaman web statis sederhana
  • Pengenalan elemen teks, gambar, link, dan struktur layout dasar
  • Latihan membuat halaman profil pribadi menggunakan HTML
  • Tips & trik menulis kode HTML yang rapi dan terstruktur
